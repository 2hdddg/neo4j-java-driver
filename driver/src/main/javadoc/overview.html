This is the reference driver implementation for Neo4j 3.0, it allows you to connect to a Neo4j server and interact with it. If you are using Java, this is generally the interface you'd use to integrate Neo4j in your application.

Installation

To include this driver in your application, you need to include it on your classpath. The driver is available as a maven artifact, and can be included in regular Maven projects as follows:

<dependency>
    <groupId>org.neo4j.driver</groupId>
    <artifactId>neo4j-java-driver</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>

Getting Connected

Once you have the driver and a connector on your classpath, you can establish sessions with Neo4j.

Have a look at the GraphDatabase class to get started.
